# KursyWalutNBP
Jak sama nazwa wskazuje, program dotyczy kursów walut NBP. Za jego pomoca można w krótkim czasie pobrać, przeanalizować oraz zapisać do pliku CSV lub XML, aktualne lub archiwalne kursy walut.
